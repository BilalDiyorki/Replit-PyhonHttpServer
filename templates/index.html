Python HTTP Server

Merhaba arkadaşlar, bu bir Python HTTP sunucusudur. Kayıt oluşturma ve giriş yapma işlemleri yapılabilmektedir. Asıl amaç RSA şifreleme yöntemini kullanarak şifreleme işlemini gerçekleştiren ve çözen bir mail uygulaması oluşturmaktır. Bu uygulama ile şifreli mail oluşturabilir ve anahtar çiftinizin public kısmını paylaşırken private kısmını saklayabilirsiniz.
